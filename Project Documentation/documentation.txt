🛒 Django E-Commerce Project Documentation
📌 Project Overview

This is a fully functional Django-based E-commerce web application built with a modular architecture. The project supports user authentication, product management, cart system, order processing, payment integration (Stripe), and review system.

The application is designed with a modern, responsive UI/UX, optimized up to 1200px screen width, ensuring a smooth experience across devices.

🧩Django Apps Structure

The project is divided into multiple apps:

accounts → User authentication, profile management
products → Product listing, details, categories
carts → Shopping cart functionality
orders → Order placement and tracking
payments → Stripe payment integration
reviews → Product review system
shared → Common utilities and shared components
